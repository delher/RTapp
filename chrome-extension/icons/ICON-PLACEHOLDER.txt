RTool Chrome Extension - Icons Needed

The extension will work without custom icons, but Chrome will show a default puzzle piece icon.

Required Icon Files:
===================

1. icon16.png  - 16x16 pixels (toolbar)
2. icon48.png  - 48x48 pixels (extensions page)
3. icon128.png - 128x128 pixels (Chrome Web Store)

How to Create:
==============

Option 1: Use existing Electron app icons
- Copy from: electron-app/build/icon.png
- Resize to 16x16, 48x48, 128x128

Option 2: Create new icons
- Design: 128x128 PNG
- Theme: Dragon emoji üêâ or "RT" text
- Colors: Purple gradient (matches popup header)
- Resize to 16, 48, 128 sizes

Option 3: Quick placeholders
Run this in terminal (requires ImageMagick):

  convert -size 128x128 xc:purple -gravity center \
    -pointsize 72 -fill white -annotate +0+0 "RT" \
    icon128.png
  
  convert icon128.png -resize 48x48 icon48.png
  convert icon128.png -resize 16x16 icon16.png

Or use an online tool like:
- favicon.io
- cloudconvert.com
- realfavicongenerator.net
